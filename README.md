# WRO Junior 2026 — Team Strg+Alt+Entf

Welcome to the official repository of **Team Strg+Alt+Entf** for the [World Robot Olympiad (WRO)](https://wro-association.org/) Junior category 2026.

## About

This repository contains all code, programs, and resources developed by our team for the WRO Junior 2026 competition.

## Team

**Team name:** Strg+Alt+Entf

## Repository Structure

| Folder / File | Description |
|---|---|
| `WRO 2026 Development.py` | Main competition program (missions, movement, color detection, and task flow) |
| `Color Calibration.py` | Utility script for calibrating HSV values with the front color sensor |
| `Line_Following_Calibration.py` | Utility script for calibrating white/black reflection threshold for line following |
| `Backup 23.05.26 1230h` | Backup copy of a previous main program version |
| `WRO2026-RM-Junior.pdf` | Official WRO 2026 Junior category rule manual |
| `WRO2026-RM-Regelwerk.pdf` | German-language WRO 2026 rules/regulations document |
| `Zeitplan Teams (Stand_ 27.05.2026).pdf` | Team schedule document (latest version in this repository) |
| `Tagesaufgaben (leaked).pdf` | Task-related reference document used during development |
| `LICENSE` | Repository license |
| `README.md` | Project overview and repository documentation |
